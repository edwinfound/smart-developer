# Smart Developer

> 一个给开发者使用的Chrome插件

## 直接安装

可以直接在Google Store安装插件

https://chrome.google.com/webstore/detail/smartdeveloper/pofpjekhccfpopfalcdjlngdfklnlacf


## 本地编译

``` bash
# 安装依赖
npm install

# 调试环境运行，启动后访问 http://localhost:8080/extension/
npm run dev

# 直接构建发布版本
npm run build
```
